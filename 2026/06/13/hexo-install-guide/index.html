从零开始搭建属于你的个人博客，这篇教程带你走完每一步。

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架，它将 Markdown 文档渲染成静态 HTML 页面。Hexo 可以在不到一秒内完成渲染，支持丰富的主题和插件系统，是目前最流行的静态博客生成器之一。

环境准备

1. 安装 Node.js

Hexo 依赖于 Node.js，请确保你的系统已安装 Node.js（推荐 v16 或以上版本）。

下载并安装：https://nodejs.org

文章格式（Front-matter）

每篇博客文章的开头都有一个 YAML 格式的 Front-matter 区域：

常用的 Front-matter 字段：

字段 说明 必填
title 文章标题 是
date 发布日期 推荐
tags 标签 否
categories 分类 否
cover 封面图片 否

编写 Markdown

Hexo 使用 Markdown 语法编写文章，支持：

  • 标题：# 一级标题、## 二级标题
  • 粗体：**粗体文本**
  • 斜体：*斜体文本*
  • 链接：[链接文字](URL)
  • 图片：![alt](图片路径)
  • 代码块：使用三个反引号包裹
  • 列表：使用 - 或 1.

常用命令速查

命令 说明
hexo new "标题" 新建文章
hexo new page "页名" 新建页面
hexo generate / hexo g 生成静态文件
hexo server / hexo s 启动本地服务器
hexo deploy / hexo d 部署到远程站点
hexo clean 清除缓存
