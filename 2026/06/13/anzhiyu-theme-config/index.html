常见问题

Q1: 主题样式没有生效？

检查以下几点：

  1. 确认 _config.yml 中 theme: anzhiyu 是否正确
  2. 运行 hexo clean && hexo generate 清除缓存
  3. 检查 themes/anzhiyu 目录是否完整

Q2: 页面加载很慢？

  • 检查图片是否过大，建议压缩到 200KB 以内
  • 使用图床服务（如 SM.MS、阿里云 OSS）存储图片
  • 考虑使用 CDN 加速静态资源
